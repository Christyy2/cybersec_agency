# cybersec_agency
Site web d'une agence de cybersécurité, créé par trois étudiants en Administrations des Systèmes et Réseaux, proposant leurs services à toutes les entreprises voulant vérifier leur sécurité numérique

## Description
Le site se compose de 4 pages :
- **Acceuil** : Présentation de l'agence Cybersec
-  **Services** : Les différents services proposés sont présentés
-  **Equipes** : Présentation de l'équipe
-  **Contact** : Formulaire de contact et coordonnées

### Technologies et Langage utilisés
- HTML5
- CSS3
- JavaScript
- Git & Github
